 1.3 Linear Algebra Kernels 2MM
 Linear algebra kernel that consists of two matrix multiplications.
 It takes the following as inputs,
α,β : scalars
 A: P*Q matrix
 B: Q*R matrix
 C: R*S matrix
 D: P*S matrix
 and gives the following as output:
 E: P*S matrix, where E = αABC+ βD
